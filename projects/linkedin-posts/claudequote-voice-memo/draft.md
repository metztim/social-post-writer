# I Built a Website While Driving My Car

**Status:** Draft
**Date:** 2026-02-12

---

I built a website while driving my car. Well, I recorded a 10-minute voice memo while driving. And that voice memo was the last meaningful creative input I gave before the site went live.

I was texting with a friend, and we were swapping some funny and surprising things Claude had said. Screenshots over WhatsApp, copy-pasted quotes. And I thought: there should be a place to collect these. Like a quote wall for the best things Claude has ever said.

So that night, driving home, I hit record and talked through the whole thing. The spec, the vibe ("Pinterest meets a Claude Code terminal"), how submissions should work (only directly from Claude Code, not copy-pasted), how Claude Code should verify whether a quote came up organically. Ten minutes of me rambling into my phone.

I gave the transcript of that voice memo to Claude Code. Here is the embarrassingly short list of everything I did after that:

1. Register the domain
2. Follow Claude Code's instructions to set up a Supabase account, paste in a script it had prepared, and provide API keys
3. Tell it to move the "how to submit" section out from under About and into its own page
4. Add DNS settings in Namecheap
5. Create a preview image with Nano Banana 3 Pro using a prompt Claude Code had written for me

I was essentially an errand boy, clicking buttons and pasting things where I was told. The entire site setup on Vercel was done by Claude Code through Vercel's command-line tool (basically a way for Claude Code to operate Vercel directly from the terminal, without me ever opening the Vercel website). No debugging. No technical problem-solving on my end.

The site is live at claudequote.com.

I tried something similar about a year ago. Built a site with Claude's help. And while it was impressive, I still had to do a lot of manual work in dashboards, configuring settings, troubleshooting deployments.

This time was different. Because of Opus 4.6. Because of Claude Code. And because tools like Vercel and Supabase have command-line tools that Claude Code can set up and operate directly. I honestly don't know if they existed before. But Claude Code knows they exist, knows how to install them, and knows how to use them. So it just... does it.

A year ago we were talking about DeepSeek, ChatGPT Pro costing $200 per month, and Claude Code didn't even exist yet. Now I can one-shot a complex production website, complete with a Claude Code plugin, from a voice memo recorded while driving my car home. 🤯

So if you haven't yet, please go check out Claude Code. It's not a weekend project. It's something you can do right now.
